# 组合数学 (combinatorics)

## 核心知识点
- **排列 (Permutation)**: 从 n 个不同元素中取 r 个有序排列，$P(n,r) = \frac{n!}{(n-r)!}$。全排列 $n!$ 种
- **组合 (Combination)**: 从 n 个不同元素中取 r 个无序选择，$\binom{n}{r} = \frac{n!}{r!(n-r)!}$
- **乘法原理**: 若任务分 k 步，第 i 步有 $n_i$ 种方式，总方式数为 $n_1 \times n_2 \times \cdots \times n_k$
- **加法原理**: 若任务有 k 种互斥方式，第 i 种有 $n_i$ 种方法，总方法数为 $n_1 + n_2 + \cdots + n_k$
- **二项式定理**: $(a+b)^n = \sum_{r=0}^n \binom{n}{r} a^{n-r} b^r$，通项 $T_{r+1} = \binom{n}{r} a^{n-r} b^r$
- **常用恒等式**: $\binom{n}{r} = \binom{n}{n-r}$；$\binom{n}{0} + \binom{n}{1} + \cdots + \binom{n}{n} = 2^n$
- **带限制的排列**: 特定元素相邻 → 捆绑法 (视为一个整体)；特定元素不相邻 → 插空法

## 高频题型

### 题型1: 基本排列组合计数
- 代表题目: 2017 P2 Q20, 2018 P1 Q5
- 解题关键: 先判断顺序是否重要 (排列 vs 组合)，再识别是否有重复元素或限制条件。密码/编号类通常用乘法原理逐位分析
- 易错点: 忘记排除重复计数；将排列当成组合 (或反之)；忽略 "at least" / "exactly" 条件的反向计数 (用补集)

### 题型2: 二项式展开与系数
- 解题关键: 写出通项公式 $T_{r+1} = \binom{n}{r} a^{n-r} b^r$，令指数等于目标次数解出 r，再代入求系数
- 易错点: 混淆项的序号 r+1 和组合数下标 r；符号处理（$(a-b)^n$ 展开带交错正负号）

## 解题策略
1. **判断类型**: 有序用 P，无序用 C，分步用乘法原理
2. **"至少"问题优先考虑补集**: P(至少1个) = 1 - P(一个都没有)
3. **二项式系数问题**: 写出通项 → 解指数方程求 r → 代入计算系数
4. **字母排列**: 先当所有字母不同排列，再除以重复字母的阶乘

## 公式速查
- $P(n,r) = \frac{n!}{(n-r)!}$，$\binom{n}{r} = \frac{n!}{r!(n-r)!}$
- $(a+b)^n = \sum_{r=0}^n \binom{n}{r} a^{n-r} b^r$
- $\binom{n}{r} = \binom{n}{n-r}$，$\sum_{r=0}^n \binom{n}{r} = 2^n$
- 有重复排列: n 个元素中有 k 种重复 (分别 $n_1, n_2, \ldots, n_k$ 个)，排列数 $= \frac{n!}{n_1! n_2! \cdots n_k!}$
