# 数列与级数 (sequences)

## 核心知识点
- **等差数列与等比数列**：通项公式 $a_n$ 与前 $n$ 项和公式 $S_n$ 的基本应用及参数关系。
- **无穷等比级数**：收敛条件（$|r| < 1$）与求和公式 $S_\infty = \frac{a_1}{1-r}$。
- **递推数列**：包括线性分式递推、函数列递推（结合微积分）以及周期性递推的通项或特定项求解。
- **指数与对数数列**：利用换元法将指数/对数方程转化为二次方程或等差/等比数列问题。
- **数列与积分的综合**：利用定积分的线性性质和牛顿-莱布尼茨公式处理由积分定义的递推数列。

## 高频题型

### 题型1: 等差与等比数列的交叉与混合
- 代表题目: 2016 P1 Q14, 2017 P1 Q7, 2020 P1 Q4, 2021 P1 Q3
- 解题关键: 设出基本量（首项、公差/公比），根据两项之差、项的对应关系或前 $n$ 项和建立方程组联立求解。
- 易错点: 遗漏等比数列公比 $r < 0$ 的情况（如 2017 P2 Q3）；混淆等差数列的下标差与公差 $d$ 的倍数关系（如 2020 P1 Q4 中第1,4,6项对应的 $d$ 倍数应为 0, 3, 5）。

### 题型2: 递推数列的规律探索与周期性
- 代表题目: 2017 P1 Q11, 2021 P2 Q20, 2022 P1 Q5
- 解题关键: 对于非标准递推式，直接计算前 3-5 项观察规律。分式递推可通过代入消元求参数；绝对值递推需结合奇偶性或分段函数特征分析。
- 易错点: 盲目尝试求通项公式而错过简单的周期性（如 2017 P1 Q11 周期为 4）；绝对值递推未正确划分区间导致积分面积计算错误。

### 题型3: 数列与微积分的综合定义
- 代表题目: 2016 P2 Q6, 2017 P1 Q17, 2017 P2 Q6, 2021 P1 Q13
- 解题关键: 先通过求导或积分化简递推式，将其转化为纯代数数列（通常是等差或等比数列），再利用求和公式求解。
- 易错点: 混淆导数递推与积分递推的效果（如 2016 P2 Q6 是求导递推导致系数变化，2017 P2 Q6 是定积分递推提取常数）；求和时上下标转换错误。
