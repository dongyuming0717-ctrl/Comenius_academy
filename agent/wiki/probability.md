# 概率 (probability)

## 核心知识点
- **古典概率**: $P(A) = \frac{n(A)}{n(S)}$，等可能样本空间中事件 A 发生的概率
- **条件概率**: $P(A|B) = \frac{P(A \cap B)}{P(B)}$，已知 B 发生下 A 发生的概率
- **独立事件**: $P(A \cap B) = P(A) \cdot P(B)$。互不影响的事件。注意：独立 $\neq$ 互斥
- **互斥事件**: $P(A \cup B) = P(A) + P(B)$ (当 $A \cap B = \emptyset$)。一般情况: $P(A \cup B) = P(A) + P(B) - P(A \cap B)$
- **补集**: $P(A') = 1 - P(A)$。"至少一个"发生用补集最简洁
- **期望值**: $E(X) = \sum x \cdot P(X=x)$。线性性质: $E(aX+b) = aE(X)+b$
- **方差**: $\text{Var}(X) = E(X^2) - [E(X)]^2$。$\text{Var}(aX+b) = a^2\text{Var}(X)$
- **二项分布**: $X \sim B(n, p)$，$P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$，$E(X)=np$，$\text{Var}(X)=np(1-p)$

## 高频题型

### 题型1: 条件概率与树状图
- 代表题目: 2016 P1 Q7, 2021 P2 Q3
- 解题关键: 画出树状图标注各分支概率。条件概率问题先写出公式 $P(A|B) = P(A \cap B)/P(B)$，分别计算分子分母
- 易错点: 混淆 $P(A|B)$ 和 $P(B|A)$ (方向反了)；条件概率的分母是条件的概率，不是全体样本空间

### 题型2: 独立性与互斥辨析
- 代表题目: 2021 P2 Q3
- 解题关键: 独立用 $P(A \cap B) = P(A)P(B)$ 验证；互斥用 $P(A \cap B) = 0$ 验证。两个概念不能混淆——互斥事件一定不独立 (除非概率为零)
- 易错点: 凭直觉判断独立性而不做数学验证；把互斥当独立

### 题型3: 期望与方差计算
- 解题关键: 先列出概率分布表 (每个取值 x 和对应概率 P(X=x))，再套公式 $E(X) = \sum xP(x)$，$E(X^2) = \sum x^2P(x)$，$\text{Var}(X) = E(X^2) - [E(X)]^2$
- 易错点: 忘记方差公式中减的是期望的平方 $[E(X)]^2$，而不是 $E(X^2)$ 再平方

### 题型4: 二项分布
- 解题关键: 确认试验满足二项分布条件 (固定次数 n、每次独立、每次只有成功/失败、成功概率 p 恒定)
- 易错点: 混淆 "at least k" ($P(X \ge k)$) 和 "exactly k" ($P(X=k)$)；忘记 $\binom{n}{k}$ 系数

## 解题策略
1. **画出样本空间或树状图** — 可视化是解概率题的最强工具
2. **"至少" → 补集**: $P(\text{至少一个}) = 1 - P(\text{零个})$
3. **条件概率先写公式**: 明确谁是条件 (竖线后面)，谁是目标事件
4. **判断独立**: 计算 $P(A \cap B)$ 是否等于 $P(A) \cdot P(B)$，不要凭感觉
5. **期望/方差**: 先建分布表，再逐项计算

## 公式速查
- $P(A|B) = \frac{P(A \cap B)}{P(B)}$
- $P(A \cup B) = P(A) + P(B) - P(A \cap B)$
- $E(X) = \sum x P(X=x)$，$\text{Var}(X) = E(X^2) - [E(X)]^2$
- 二项分布 $B(n,p)$: $P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$，$E(X)=np$

> 共 4 道真题，覆盖古典概率、条件概率与期望。概率题常与组合计数联合考察。
